HTML

HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages HTML describes the structure of Web page. HTML consists a series of elements. HTML elements tell the browser how to display the content.

PHP

PHP is an open-source, server-side programming language that can be used to create websites, applications, customer relationship management systems and more. It is a widely-used general-purpose language that can be embedded into HTML.

CSS

What is CSS? CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work. It can control the layout of multiple web pages all at once.
